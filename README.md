# 💣 Boom — классическая аркадная игра в стиле Bomberman

**Версия:** 1.0.0  
**Unity:** 2021.3+  
**Платформы:** Windows, macOS, Linux

**Boom** — это ремейк культовой аркадной игры Bomberman, разработанный на языке C# с использованием игрового движка Unity. Игроку предстоит стратегически размещать бомбы, уничтожать врагов, собирать бонусы и проходить уровни, избегая собственных взрывов.

## О проекте

**Boom** — это классическая аркадная игра, вдохновлённая легендарной Bomberman. Проект был разработан в рамках курсовой работы и представляет собой полностью функциональную игру с несколькими уровнями, системой бонусов и интеллектуальными врагами.

Игрок управляет персонажем, который может устанавливать бомбы для разрушения стен и уничтожения врагов. По мере прохождения уровня можно собирать бонусы, улучшающие характеристики персонажа. Цель каждого уровня — уничтожить всех врагов и не погибнуть от собственных взрывов.

## Геймплей

### Основная цель
Уничтожить всех врагов на уровне, используя стратегически размещённые бомбы, и не погибнуть от взрывов.

### Игровой процесс
- Перемещайтесь по лабиринту, разрушайте стены с помощью бомб
- Собирайте бонусы для улучшения способностей
- Избегайте врагов и собственных взрывов
- Проходите уровни с возрастающей сложностью

## Бонусы

В игре представлено **7 видов бонусов**, которые выпадают при разрушении некоторых стен:

| Иконка | Бонус | Эффект |
|:------:|-------|--------|
| 💣 | **Дополнительная бомба** | +1 к количеству одновременно установленных бомб |
| 🔥 | **Огонь** | Увеличение радиуса взрыва |
| ⚡ | **Скорость** | Увеличение скорости передвижения |
| 🧱 | **Прохождение сквозь стены** | Временная неуязвимость к стенам |
| 🔥 | **Неуязвимость к огню** | Защита от собственных взрывов |
| 💣 | **Неуязвимость к бомбам** | Прохождение сквозь бомбы |
| 🎛️ | **Детонатор** | Возможность подрывать бомбы по команде |

> Бонусы с временным эффектом действуют **30 секунд**.

## Установка и запуск

### Из исходного кода

```bash
# Клонируйте репозиторий
git clone https://github.com/a1inas/Boom-Game.git

# Перейдите в папку проекта
cd Boom-Game
```

1. Установите **Unity 2021.3** или новее
2. Откройте проект через **Unity Hub**
3. В окне Unity откройте сцену `Scenes/MainMenu.unity`
4. Нажмите кнопку **Play** для тестирования

### Готовая сборка

1. Скачайте последний релиз из раздела [Releases](https://github.com/a1inas/Boom-Game/releases)
2. Распакуйте архив
3. Запустите `Boom.exe` (Windows) или соответствующий файл для вашей ОС

## Ключевые механики

### Алгоритм поиска пути (A*)
Враги используют алгоритм **A\*** для нахождения оптимального пути к игроку, учитывая препятствия и разрушаемые стены. Это обеспечивает интеллектуальное поведение противников и повышает сложность игры.

### Система бонусов
При разрушении стен с вероятностью выпадают бонусы. Система реализована через компонент `PowerUp`, который определяет тип бонуса и его эффект при контакте с игроком.

### Взрывная волна
При взрыве бомбы огонь распространяется в четырёх направлениях. Дальность взрыва зависит от текущего уровня огня игрока, который можно увеличивать с помощью бонусов. Взрыв уничтожает:
- Врагов
- Разрушаемые стены (Brick)
- Может убить самого игрока

### Умная камера
Камера автоматически следует за игроком, но ограничена границами уровня. Это реализовано через компонент `CameraController`, который вычисляет допустимые границы движения на основе расположения стен.
